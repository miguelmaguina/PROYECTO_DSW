TAREA 1 - DIFERENCIA ENTRE APP Y APLICACIÓN WEB

Tabla 1. Cuadro Comparativo.

Característica Aplicación de escritorio Aplicación Web
Fiabilidad La aplicación de escritorio funciona de forma autónoma, por lo que lo principal es la calidad del código y la estabilidad del hardware sobre el que se ejecuta este código. El funcionamiento de una aplicación web depende no solo de la competencia que se desarrolle y de las características del dispositivo del usuario, sino también de la velocidad de la conexión a Internet y del rendimiento del servidor remoto.
Desempeño Las aplicaciones de escritorio suelen estar desarrolladas pensando un sistema operativo y familia hardware concretos. El servidor web es el que ejecuta la mayor parte del código de la aplicación, y suministra de forma centralizada las vistas (las páginas) a los navegadores conectados.

Descarga Una de la principales diferencia que tienen la aplicaciones de escritorio es que necesitan descargarse, lo cual puede ser un problema para el usuario ya que consume recursos de su dispositivo, así como puede necesitar de complementos adicionales para su correcto funcionamiento. En cuanto a las aplicaciones web, estas no requiere de una descarga pues son una versión optimizada de las páginas web y que ya usa todos los recursos que le proporciona el navegador desde el cual se ejecuta. Po ello, su practicidad de uso muchas veces la hace ser la forma más usada de poder brindar un servicio o producto.
Actualización Otra de sus grandes diferencias de las aplicaciones de escritorio es que necesitan que el usuario las actualice constantemente para disfrutar de las novedades que se realice en la app, esto puede tomar mucho tiempo. En las aplicaciones web necesitan ser actualizadas por el desarrollador y el usuario no tiene que realizar nada para actualizarlas porque ya lo están cuando accede a la plataforma.
Seguridad Las aplicaciones de escritorio le permiten al usuario almacenar sus datos privados de manera local sin conexión en lugar de en la nube, característica que puede ser considerada tanto positiva como negativa. La información contenida en las aplicaciones web se almacena en la nube y no en discos duros físicos. Evitando el riesgo latente de que dicha información se pierda por un fallo del dispositivo, por una pérdida o un virus.
